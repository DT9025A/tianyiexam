## 编写环境 : AIDE & JAVA

## 使用方法
- 安装APK，API<13/IOS/WINDOWS 是不能安装的哟
- 点击桌面上的图标
- 点击 "查询" 来获取考试信息
> example: 高三二连Q
- 输入考生号 (可与上一步对调)
- 点击 "查询" 进行信息查询
- 捕获 java.io.FileNotFoundException (划掉)
- 查询完成
- 该自闭的自闭，改开心的开心，该膨胀的膨胀
- 退出程序
